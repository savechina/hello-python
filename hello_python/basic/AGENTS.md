# hello_python.basic

**Purpose:** Foundational Python concepts — datatypes and expressions. Entry-level tutorials.

## STRUCTURE
```
basic/
├── __init__.py
├── datatype_sample.py     # Basic types, strings, collections
└── expression_sample.py   # Operators, expressions, control flow
```

## WHERE TO LOOK
| Concept | File |
|---------|------|
| Types, strings, lists, dicts | `datatype_sample.py` |
| Operators, expressions | `expression_sample.py` |

## CONVENTIONS
- Functions are standalone, not classes
- Runnable as `python basic/<file>.py` via `if __name__ == "__main__"` blocks
- Test class naming: `Test<Feature>Sample` (e.g., `TestExpressioneSample` — note typo with extra 'e')
